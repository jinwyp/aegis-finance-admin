第六条  有关交易的特别约定

6.1 卖方应承担缴纳煤炭场地交货前所有费用的特别约定

本合同标的煤炭在交货港港场地交货前所有的流转费用（包括但不限于商检费、船运费、滞期费、海关税费、装卸费、堆存费等）均应由卖方承担并支付，在卖方向买方办理了煤炭所有权转移手续后仍存在卖方未缴纳的费用，卖方负有向有关方缴纳的义务，若相关收费方催收，卖方应立即缴纳。买方为了保障自身权益而不得不代为缴纳时，买方保留向卖方追索的权利。若因卖方未及时缴纳有关费用造成买方不能自由交提货物，视为卖方违约。由此给买方造成损失的，卖方应承担完全责任。

6.2 买方对本合同标的煤炭质、数量验收责任的特别约定

在买卖双方交易、买方向下游销售合同项下煤炭货物过程中，对煤炭的质、数量验收责任均由卖方承担，卖方负责协调交货港或其他有关单位进行煤炭的装卸、化验及堆存等事宜，买方不承担本合同标的煤炭质、数量验收责任，亦不承担因质、数量验收而产生的任何费用。

6.3 其他的特别约定

（1）卖方保证其向买方转移的煤炭所有权合法、完整、无瑕疵，即卖方保证其销售给买方的本合同标的煤炭拥有完整的、合法的、无可争辩的所有权，且不存在抵押、留置等任何第三方权利。如遇货物的有关权利（包括但不限于所有权、知识产权）而引起的有关争议，卖方应负责解决并承担由此产生的费用，并赔偿由此给买方造成的所有损失，买方拥有向卖方追索已付货款本金、资金利息（按照日万分之五计息）及其它一切损失的权利。

（2）合同项下货物在港口存在灭失风险，买卖双方特别约定卖方将煤炭所有权转移给买方后，至买方销售完该批货物前，本合同项下煤炭货物的灭失等风险仍由卖方承担全部责任。
